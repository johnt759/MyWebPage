CPSC 120: Introduction to Programming

Course Summary: Introduction to the concepts underlying all computer programming: design and execution of programs; sequential nature of programs; use of assignment, control and input/output statements to accomplish desired tasks; design and use of functions. Structured and object-oriented methodologies.

  • Start Date: August 2016
  • End Date: December 2016
  • Programming language(s): C++

List of skills learned:

  • C++ Programming
  • Pseudocode Design
  • Data Types
  • Constants and Variables
  • Relational and Logical Operators
  • Loops and If-Else Conditions
  • Arrays
  • Function Declarations
