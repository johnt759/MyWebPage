CPSC 535: Advanced Algorithms

Design and analysis of sophisticated algorithms and data structures. Lower, tightand pseudo-polynomial bounds. Randomization, approximationand special-purpose data structures. Algorithmic frameworks, such as maximum flow and linear programming. CPSC 335 recommended.

  • Start Date: August 2022
  • End Date: December 2022
  • Programming language(s): Java, Python

List of skills learned:

  • Asymptotic Notation
  • Linear-Time Sorting
  • Stacks, Queues, and Linked Lists
  • Hashing and Open Addressing
  • Binary Search Trees
  • Dynamic Programming and Greedy Methods
  • Minimum Spanning Trees
  • Single-Source Shortest Paths
  • Maximum Flow
  • Linear Programming
  • String Matching
  • NP-Completeness
