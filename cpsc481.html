CPSC 481: Artifical Intelligence

Using computers to simulate human intelligence. Production systems, pattern recognition, problem solving, searching game trees, knowledge representation and logical reasoning. Programming in AI environments.

  • Start Date: August 2020
  • End Date: December 2020
  • Programming language(s): Python

List of skills learned:

  • Concepts of Artificial Intelligence
  • AI Agents
  • Breadth-first and Depth-first searching
  • Heuristic and A* searching
  • Hill-climbing search
  • Local optimization
  • Genetic algorithms
  • Simulated Annealing
  • Game theory
  • Minimax search
  • Alpha-Beta pruning
  • Propositional logic
  • Bayesian networks
