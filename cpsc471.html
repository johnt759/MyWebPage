CPSC 471: Computer Communications

Introduction to digital data communications. Terminology, networks and their components, common-carrier services, telecommunication facilities, terminals, error control, multiplexing and concentration techniques.

  • Start Date: January 2020
  • End Date: May 2020
  • Programming language(s): Python

List of skills learned:

  • Networking Concepts
  • Packet and Circuit Switching
  • Throughputs, Queing, and Delays
  • TCP/IP and OSI models
  • Application Layer
  • HTTP, SMTP, and DNS
  • Transport Layer
  • TCP vs UDP
  • Congestion Control
  • Network Layer
  • Routers and Packets
  • Internet Protocol Addressing
  • Routing Algorithms
  • Link Layer
  • Checksum and Cyclic Redundancy Checking
  • Local Area Networks
